\abstracttexten{This master thesis deals with analysis of descriptions of semantic contracts in Java technologies. Main purpouse of this diploma is creation of a tool which enables extraction of chosen constructions of design by contract which is part of semantic contracts. To be able to create the tool it is firstly necessary to design model which enables to store represenations of various contracts. First part of this thesis is dedicated to theoretical introduction to contracts espicially design by contract and then to analysis of language Java from the point of grammer and tokenization. Second part contains information about implemention of the tool including the design of model and results of this work.}

\abstracttextcz{Tato diplomová práce se zabývá analýzou popisů sémantického kontraktu v Java technologiích. Hlavní náplní práce je tvorba nástroje, který umožní extrakci vybraných konstrukcí design by contract, které se řadí do kategorie sémantických kontraktů. Aby bylo možné daný nástroj vytvořit je nejprve nutné navrhnout model, který umožní zachytit reprezentaci různých kontraktů. První část práce je věnována teoretickému úvodu do problematiky kontraktů, zejména pak design by contract a následně rozboru jazyka Java z hlediska gramatiky a tokenizace. Druhá část pak obsahuje informace o implementaci daného nástroje, společně s návrhem modelu a dosaženými výsledky.}
